# ikuai-openvmtools

适用于爱快系统的open-vm-tools，可以实现运行在ESXi上的爱快接收vmtools关机信号并正常关机

---

## 环境变量

| 变量名称    | 默认值    | 描述
| :-------- | :-----   | :-----
| PORT | 80|爱快端口号|
| USERNAME | admin | 爱快用户名
| PASSWORD | admin | 爱快密码
